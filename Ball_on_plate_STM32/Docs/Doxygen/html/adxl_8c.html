Detailed Description

En este archivo se encuentran las funciones de manejo del acelerometro. IMPORTANTE No zarparse con la velocidad de SPI, el maximo segun la hoja de datos es 4MHz, pero ya si superas 1,6MHz tenes que empezar a poner delays entre datos de la trama los cuales son un quilombo y no es practico. O sea, no pongas una velocidad mayor de 1,6MHz si no queres sufrir como sufri yo.
